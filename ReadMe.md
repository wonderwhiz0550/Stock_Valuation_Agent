\# Agentic Stock Valuation Platform



An AI-powered equity research platform combining:



\- Multi-stage DCF

\- Monte Carlo Simulation

\- SEC Filing Retrieval

\- FAISS-based RAG

\- GPT-4o Investment Thesis Generation

\- Streamlit Frontend



\---



\## Features



\### Valuation Agent



Performs:



\- Multi-stage DCF

\- Monte Carlo Simulation

\- Fair Value Estimation



\---



\### News Agent



Retrieves:



\- Latest Company News

\- Basic Sentiment Analysis



\---



\### RAG Agent



Automatically:



\- Downloads SEC Filings

\- Builds FAISS Index

\- Retrieves Relevant Context



\---



\### Thesis Agent



Uses GPT-4o to generate:



\- Executive Summary

\- Growth Drivers

\- Risks

\- Bull Case

\- Bear Case

\- Final Assessment



\---



\## Installation



```bash

git clone <repo>



cd stock-valuation-ai



pip install -r requirements.txt
```
